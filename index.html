Rethink DB

Overview

  • Opensource NoSQL database
  • For a realtime web apps
  • Stores JSON
  • Uses ReQL (DSL)
  • Client-side triggers through changefeeds & notifications

Who uses it?

  • Extra-Vehicular Activity - spacesuits

  • Instant messaging in online courses

  • Collaboration solutions for business

  • Data collection and curation process

Horizon & RethinkDB

Horizon

  • Enables to build realtime apps without writing backend code
  • Created by Rethink DB
  • Opensource
  • Contains node.js ecosystem
  • Can be used as library to node.js app
  • Contains components: subsriptions, persistence, authentication, sessions, etc.
  • Scals lineary by Rethink DB's clustering engine
  • Queries return RxJS Observables
